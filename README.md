# 🏢 بوت تحليل الأسهم - النسخة المتكاملة

## 📖 نظرة عامة
تطبيق تحليل أسهم متطور يدعم **5 أسواق مالية** مختلفة مع أكثر من **100 سهم** وميزات تحليل ذكية باستخدام الذكاء الاصطناعي.

### 🌍 الأسواق المدعومة
- 🇪🇬 **البورصة المصرية (EGX)** - 40+ سهم
- 🇸🇦 **تداول السعودية (Tadawul)** - 30+ سهم
- 🇦🇪 **سوق أبوظبي للأوراق المالية (ADX)** - 6 أسهم
- 🇦🇪 **سوق دبي المالي (DFM)** - 5 أسهم
- 🇺🇸 **الأسهم الأمريكية (NASDAQ & NYSE)** - 35+ سهم

## ⚙️ طريقة التشغيل

### 1️⃣ تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2️⃣ إعداد مفتاح Gemini AI (اختياري)
إذا كنت تريد استخدام ميزة **التحليل الذكي بالذكاء الاصطناعي**:

1. احصل على مفتاح API مجاني من [Google AI Studio](https://makersuite.google.com/app/apikey)
2. أنشئ ملف `.streamlit/secrets.toml` في مجلد المشروع:
```toml
GEMINI_API_KEY = "ضع_مفتاحك_هنا"
```

### 3️⃣ تشغيل التطبيق
```bash
streamlit run app.py
```

سيفتح التطبيق تلقائياً في المتصفح على `http://localhost:8501`

## 🎯 الميزات الرئيسية

### ✨ التحليل الفني
- **مؤشر RSI** - تنبيهات تلقائية لمناطق التشبع
- **المتوسط المتحرك (SMA)** - متابعة الاتجاهات
- **رسوم بيانية تفاعلية** - باستخدام Plotly
- **بيانات فورية** - من Yahoo Finance

### 🔍 البحث المتقدم
- بحث بالاسم العربي أو الإنجليزي
- بحث برمز السهم
- فلترة حسب السوق

### 🤖 الذكاء الاصطناعي
- تحليل ذكي بواسطة **Google Gemini**
- توصيات استثمارية
- تحليل الدعم والمقاومة

## 📂 هيكل الملفات

```
stock-analyzer/
├── database.py       # قاعدة بيانات الأسهم (100+ سهم)
├── core.py          # المدير الذكي للنظام
├── app.py           # التطبيق الرئيسي (Streamlit)
└── requirements.txt # المكتبات المطلوبة
```

## 🔧 إضافة أسهم جديدة

لإضافة أسهم جديدة، افتح ملف `database.py` وأضف السهم في السوق المناسب:

```python
"اسم الشركة": "TICKER.SUFFIX",
```

مثال:
```python
"البنك الجديد": "NEWB.CA",  # للبورصة المصرية
"شركة سعودية": "1234.SR",   # لتداول السعودية
```

## 📊 مثال على الاستخدام

1. افتح التطبيق
2. اختر تبويب "🔍 بحث عن سهم" أو "📋 تصفح جميع الأسهم"
3. ابحث عن السهم المطلوب (مثال: "الراجحي" أو "Apple")
4. اختر الفترة الزمنية للتحليل
5. اضغط على "🤖 تحليل ذكي" للحصول على توصيات AI

## ⚠️ ملاحظات هامة

- البيانات المالية من **Yahoo Finance** - تأكد من الاتصال بالإنترنت
- التحليل الذكي يتطلب مفتاح **Gemini API**
- البيانات للأغراض التعليمية فقط - **ليست نصيحة استثمارية**

## 🚀 نشر التطبيق على Streamlit Cloud

1. ارفع الملفات على GitHub
2. اذهب إلى [Streamlit Cloud](https://streamlit.io/cloud)
3. اربط الريبو واختر ملف `app.py`
4. أضف `GEMINI_API_KEY` في إعدادات Secrets
