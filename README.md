# 🏢 بوت تحليل الأسهم - النسخة المتكاملة

## 📊 نظرة عامة

بوت متقدم لتحليل الأسهم يدعم أسواق متعددة:
- 🇪🇬 البورصة المصرية (EGX)
- 🇸🇦 تداول السعودية (Tadawul)
- 🇦🇪 سوق أبوظبي للأوراق المالية (ADX)
- 🇦🇪 سوق دبي المالي (DFM)
- 🇺🇸 الأسهم الأمريكية (NASDAQ & NYSE)

## ✨ المميزات

- ✅ دعم أكثر من 100+ سهم من 5 أسواق مختلفة
- ✅ بحث ذكي عن الأسهم بالاسم أو الرمز
- ✅ تحليل فني متقدم (RSI, SMA, Charts)
- ✅ تنبيهات ذكية للمستثمرين
- ✅ تكامل مع Google Gemini AI للتحليل الذكي
- ✅ واجهة سهلة الاستخدام بالعربية

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

### 2. (اختياري) إعداد مفتاح Gemini API

قم بإنشاء ملف `.streamlit/secrets.toml` وأضف مفتاح API:

```toml
GEMINI_API_KEY = "your_api_key_here"
```

للحصول على مفتاح API مجاني، قم بزيارة: https://makersuite.google.com/app/apikey

### 3. تشغيل التطبيق

```bash
streamlit run app.py
```

## 🔧 كيفية إضافة أسهم جديدة

1. افتح ملف `database.py`
2. أضف السهم الجديد في السوق المناسب:

```python
"اسم السهم": "SYMBOL.SUFFIX",
```

مثال:
```python
"بنك الإمارات دبي الوطني": "ENBD.DU",
```

## 📊 المؤشرات الفنية المدعومة

- **RSI (Relative Strength Index)**: قياس قوة الاتجاه
- **SMA (Simple Moving Average)**: المتوسط المتحرك البسيط
- **رسوم بيانية تفاعلية**: باستخدام Plotly
