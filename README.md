# 🏢 بوت تحليل الأسهم - النسخة المعمارية المتكاملة

## 📋 الوصف
تطبيق تحليل أسهم احترافي يدعم الأسواق العربية والعالمية مع ذكاء اصطناعي متقدم.

## 🌍 الأسواق المدعومة
- 🇪🇬 البورصة المصرية (EGX) - 40+ سهماً
- 🇸🇦 تداول السعودية (Tadawul) - 30+ سهماً
- 🇦🇪 سوق أبوظبي (ADX) - 6 أسهم
- 🇦🇪 سوق دبي المالي (DFM) - 5 أسهم
- 🇺🇸 الأسهم الأمريكية (NASDAQ & NYSE) - 40+ سهماً

## ✨ المميزات
- ✅ **بحث ذكي**: ابحث عن الأسهم بالاسم أو الرمز
- ✅ **تحليل فني متقدم**: RSI, SMA, والمؤشرات الفنية
- ✅ **رسوم بيانية تفاعلية**: باستخدام Plotly
- ✅ **تنبيهات ذكية**: إشعارات للمناطق المهمة
- ✅ **تحليل بالذكاء الاصطناعي**: باستخدام Google Gemini
- ✅ **دعم متعدد الأسواق**: فلترة حسب السوق
- ✅ **بنية معمارية قابلة للتوسع**: سهولة إضافة أسهم جديدة

## 🚀 التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. إضافة مفتاح Gemini API (اختياري)
قم بإنشاء ملف `.streamlit/secrets.toml`:
```toml
GEMINI_API_KEY = "your-api-key-here"
```

### 3. تشغيل التطبيق
```bash
streamlit run app.py
```

## 📊 كيفية الاستخدام

### البحث عن سهم
1. انتقل إلى تبويب "🔍 بحث عن سهم"
2. اكتب اسم السهم أو رمزه
3. اختر السهم من النتائج

### تصفح الأسهم
1. انتقل إلى تبويب "📋 تصفح جميع الأسهم"
2. اختر السوق من القائمة المنسدلة
3. اختر السهم

### التحليل
- عرض الرسوم البيانية التفاعلية
- مراجعة المؤشرات الفنية (RSI, SMA)
- قراءة التنبيهات الذكية
- استخدام زر "🤖 تحليل ذكي" للحصول على توصيات

## 🔧 إضافة أسهم جديدة

لإضافة أسهم جديدة، قم بتعديل ملف `database.py`:

```python
"TADAWUL": {
    "stocks": {
        "اسم السهم الجديد": "TICKER.SR",
        # ... المزيد من الأسهم
    }
}
```

## 📈 المؤشرات المدعومة
- **RSI (Relative Strength Index)**: قياس قوة السهم
- **SMA (Simple Moving Average)**: المتوسط المتحرك البسيط
- **تنبيهات التشبع الشرائي/البيعي**: عند RSI > 70 أو < 30
